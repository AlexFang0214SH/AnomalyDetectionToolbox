This data set is modified from NASA Shuttle Valve Data 
(http://cs.fit.edu/~pkc/nasa/data/). Normal sequence 
data set includes TEK00000 through TEK00002. Unseen 
sequence data set TEK00010 through TEK00017 as 
abnormal, and TEK00003 as normal.
