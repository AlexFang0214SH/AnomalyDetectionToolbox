• Algorithm: Heuristically Ordered Time series using Symbolic Aggregate ApproXimation
  • Authors: Eamonn Keogh, Jessica Lin, Ada Fu
  • Original paper: HOT SAX: Efficiently Finding the Most Unusual Time Series Subsequence
  • Website: http://www.cs.gmu.edu/~jessica/sax.htm
  • Description: HOTSAX is good at finding an pattern discord (or we call it anomaly) in a single timeseries data. A input time series data will be transfromed into series of SAX symbols and compare with each other to find the discard. It use outer and inner heuristic to speed up the algorithm.
