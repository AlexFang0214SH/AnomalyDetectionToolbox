Oversampling Principal Component Analysis

Reference: Lee, Yuh-Jye; Yeh, Yi-Ren; Wang, Yu-Chiang Frank, "Anomaly Detection via Online Oversampling Principal Component Analysis," Knowledge and Data Engineering, IEEE Transactions on , vol.25, no.7, pp.1460,1470, July 2013 Description: Oversampling PCA determines the degree to which a data instance is an outlier by taking advantage of PCA's sensitivity to outliers. The target instance is oversampled and a principal component analysis is run on the data set. If there is significant perturbation in the leading eignevector as compared to that returned by running PCA on the original data set, then the target instance is considered an outlier. Low computation complexity and quick updating of the principal directions make this method of interest for online learning.
