• Algorithm: fast Angle Based Outlier Detection
  • Authors: Hans-Peter, Kriegel Matthias, Schubert Arthur Zimek
  • Original paper: Angle-Based Outlier Detection in High-dimensional Data In KDD2008
  • Website: http://www.dbs.ifi.lmu.de/
  • Time Complexity: O(n^3)
  • Description: A approximation version of ABOD. In ABOD method, an ABOF of each point uses information from each pair in all other points. But fast ABOD only uses information from eack pair in other K points.
